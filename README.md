# py-uni-practice
Simple editor in Python + Raylib

# Run raw
## Linux
```bash
pip install raylib numpy

# In repo directory

./main.py
```

## Windows
```shell
pip install raylib numpy 

# In repo directory

./main.py
```

# Build
## Linux
```bash
# In repo directory

pip install pyinstaller
python3 build.py

# OUTPUT: dist/imgjoy
```

## Windows
```shell
# In repo directory

pip install pyinstaller
python3.exe build.py

# OUTPUT: dist/imgjoy.exe
```
