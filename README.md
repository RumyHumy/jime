# Jime
- **J**oyful **im**age **ed**itor written in python with raylib

# Run raw
## Linux
```bash
pip install raylib numpy

# In repo directory

./main.py
```

## Windows
```shell
pip install raylib numpy 

# In repo directory

./main.py
```

# Build
## Linux
```bash
# In repo directory

pip install pyinstaller
python3 build.py

# OUTPUT: dist/imgjoy
```

## Windows
```shell
# In repo directory

pip install pyinstaller
python3.exe build.py

# OUTPUT: dist/imgjoy.exe
```
